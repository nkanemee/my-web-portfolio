Sunny Acres U-Pick Farm

Role: UI/UX Designer & Frontend Developer

Tools: Figma, HTML5, CSS Grid, Adobe Illustrator, GitHub

Overview

Sunny Acres U-Pick Farm is a conceptual responsive website designed to promote a local farms seasonal produce and events. The layout highlights available crops, visitor information, and booking options for community events such as hayrides and pumpkin picking.

My Contribution

  • Designed a bright, nature-inspired color palette and typography system in Figma
  • Built a responsive homepage using CSS Grid and Flexbox
  • Created reusable UI components for product cards, navigation, and event listings
  • Implemented a mobile-first layout prioritizing accessibility and touch-friendly CTAs

What I Learned

Gained practical experience designing a responsive site using both visual tools and code. Focused on enhancing user experience for outdoor and local business audiences by using thematic imagery, clear hierarchy, and action-oriented layout patterns.
