Job Search App

Role: Full-Stack Developer

Tools: JavaScript (ES6), Bootstrap, Node.js, Express.js, REST API, Fetch API, async/await, GitHub

Overview

This app enables users to search and view job listings pulled from a custom-built Node.js/Express backend API. The interface dynamically fetches data based on user input and renders job listings using modern JavaScript ES6 modules.

My Contribution

  • Built and deployed a backend REST API for job data retrieval
  • Developed a modular, responsive frontend with Bootstrap and Parcel bundler
  • Implemented form validation, async/await fetch requests, and reusable UI components
  • Integrated smooth DOM updates and loading animations for improved UX

What I Learned

Enhanced my ability to write clean, reusable JavaScript modules and improved proficiency in working with REST-based APIs. Also gained real-world experience managing front-end builds and version control with GitHub.
